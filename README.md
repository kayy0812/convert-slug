# Convert PlainText To Slug Text
## How to use?
### Module import
```
import convertSlug from 'convert-slug';
```

### Common import
```
const convertSlug = require('convert-slug');
```

### Convert function
```
const text = 'I love you 3000';
const slug = convertSlug(text);

console.log(slug);
```
